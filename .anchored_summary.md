## Progress

### Done — Bug Fixes & Enhancements
- ✅ **Fixed duplicate Kids Zone nav button** — removed first `navSetup` override that also added `navKids`; kept second override which adds `navKidsZone` + `navProfiles`
- ✅ **Fixed welcome buttons duplicating** — added `.extra-buttons` guard class to prevent re-adding buttons on every `showWelcome()` call
- ✅ **Fixed matching game feedback timing** — cleared old feedback on new selection, added `display:block` inline, extended wrong animation to 1s, moved `speakWord` after feedback
- ✅ **Fixed PWA desktop caching** — synced `sw.js` version (`app.js?v=11`) with `index.html`, bumped cache `opencode-v8` → `opencode-v9`
- ✅ **Wired Level Test to UI** — added "🧪 اختبار المستوى" button in level header (`renderTOC`) and dashboard progress items
- ✅ **Fixed Level Test hardcoded index** — `showLevelTest(li)` now accepts level index parameter; reads CEFR level from curriculum data; `submitLevelTest` saves result to correct level index
- ✅ **Fixed Placement Test thresholds** — changed from hardcoded percentages (`<40→A1...`) to dynamically using the `scoring` object from `placement_test.json` (A1:0-8, A2:9-16, B1:17-24, B2:25-32, C1:33-37, C2:38-40)
- ✅ **Fixed `checkQuiz` comparison** — now compares by option index (`data-ans-idx`) instead of `textContent`, with text fallback for legacy data
- ✅ **Enhanced `evaluateWriting`** — added word-match check against model answer, showing match percentage and key vocabulary hints
- ✅ **5 new features added** under `➕` button:
  - 💡 Daily Word — random vocab word daily
  - 🧩 Sentence Scramble — arrange shuffled words
  - 🐝 Spelling Bee — TTS + spelling with 3 attempts
  - 📊 Progress Charts — bar chart + level completion bars
  - 📄 Worksheets PDF — printable worksheet per lesson

## App Structure
- `app.js` (~2565 lines) — all features + fixes
- `index.html` — HTML with `app.js?v=11`
- `style.css` — styles + confetti keyframes
- `server.js` — Node.js HTTP server
- `sw.js` — Service Worker (cache `opencode-v9`)
- `placement_test.json` — 40 Cambridge-style questions (A1-C2)
- `level_tests.json` — level-specific tests per CEFR level
- `package.json` — Node >=18, start script
